# Previsao-de-Estoque-Inteligente
Este projeto teve como objetivo o desenvolvimento de um Modelo de Machine Learning para Previsão de Estoque Inteligente utilizando o Python.

# Tecnologias utilizadas
- Python
- Numpy
- Pandas
- Sklearn.

  # Funcionamento
Ele não prevê vendas diretamente. Ele aprende o consumo real observando a queda do estoque ao longo do tempo.
O modelo ler e analisa o Dataset e com base nisso, ele realiza a previsão do Consumo de Estoque.
